# Node Js&npm学习

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时/运行环境。

npm是随同NodeJS一起安装的包管理工具。

## Node Js

1.下载地址：

https://nodejs.org/en/download/

2.环境变量：

```
D:\nodejs\install         #nodejs路径

D:\nodejs\install\node_modules\npm\bin
                          #npm路径
```

3.查看版本信息：

```
node -v            #查看版本
npm -v             #npm版本
```

4.运行：

```
node file_name.js
```

## npm

常用命令：

```
npm install       #安装依赖环境
npm run dev       #运行前端
npm run build     #打包
```
