## Spring Booot + Vue 开发Java Web
My Todo Lists
### 开发环境介绍
* JDK1.8
* Node v10.16.3
* npm v6.9.0
* 开发工具IDEA（安装Vue.js插件）
* 数据库MySQL v8.0.18
* 版本管理工具 Git

## 快速开始

git clone 或 下载项目  
后端：  
通过IDEA打开直接运行项目  
或者  
```
java -jar [jar包]
```
前端：  
项目目录下命令行：  
```
npm install
npm run dev
```
前端访问localhost:8080,后端端口为localhost:8888
